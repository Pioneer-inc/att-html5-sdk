API Platform SDK for HTML5

What is the AT&T API Platform SDK for HTML5?

The AT&T API Platform SDK for HTML5 is designed to provide an easy way to develop Web applications that can access the AT&T APIs using JavaScript. The HTML5 experience for AT&T developers focuses on an enhanced capability to leverage the AT&T network-based services to create feature-rich apps. Access to services like SMS, Speech-to-Text, and MMS, in conjunction with the ability to enable direct carrier billing, provides a multitude of options not normally available to mobile Web application developers.

Dependencies

This SDK provides a framework-neutral library (att-api-client.js) for accessing the AT&T services. However, all the sample applications require Sencha Touch (version 2.3.1 or later). After agreeing to the Sencha software license, download the Sencha Touch and copy the following files into the webcontent/lib folder:


sencha-touch-all.js

sencha-touch.css


If you choose to run the Java variant of the included SDK server, you must recompile it so that the resulting .war file includes the Sencha Touch files listed above.
