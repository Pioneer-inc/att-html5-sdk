Sample Applications

These Sample Applications demonstrate how to use the AT&T API Platform SDK for HTML5. They are all developed using Sencha Touch and they can access the AT&T APIs using javascript.

What is Sencha Touch?

Sencha Touch allows you to develop mobile web apps that look and feel native on iPhone, Android, and BlackBerry touch devices. If you have not developed a mobile application using Sencha Touch or are not familiar with HTML5 technologies please visit Sencha Learn where we provided dozens of tutorials and walkthroughs for Sencha Touch and HTML5 http://www.sencha.com/learn/touch/ for more information.
